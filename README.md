# Login Authentication System

A Java Web Application for user authentication built using **JSP, Servlets, and MySQL**.

## Features
- User Registration
- User Login
- Forgot Password
- Reset Password
- MySQL Database Integration
- Styled UI using HTML/CSS

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat
- HTML / CSS

## Database

Database Name: `loginapp`

Table: `users`

| Column   | Type         |
|---------|--------------|
| id      | INT (PK)     |
| username| VARCHAR(50)  |
| email   | VARCHAR(100) |
| password| VARCHAR(100) |

## Setup Instructions

1. Clone the repository
2. Import project into Eclipse
3. Configure MySQL database
4. Add MySQL Connector JAR
5. Run the project on Apache Tomcat
